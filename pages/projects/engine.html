OPENGL ENGINE

I've been working on an OpenGL engine as the final project for a Data Structures class. A basic outline of how to complete the engine was provided, but most of the code was left up to students to figure out for themselves. Design choices were also left up to the student, allowing for many different projects running many different games. For my game, I went with a small bowling simulation, implementing collision detection and resolution, model loading, and lighting, along the way. This engine will certainly continue to be a work-in-progress for some time, with one immediate goal being to swap out the current instructor-provided models with my own models. If you'd like to take a look at it in Visual Studio 2015, a download is provided below.
