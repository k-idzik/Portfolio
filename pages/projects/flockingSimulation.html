FLOCKING SIMULATION

I created this flocking kitten simulation in Unity as a final project for an Interactive Media class, and while most of the code is my own, some scripts were provided to me by the instructor. Transitioning from 2D to 3D, I used models built into Unity or from the Unity Asset Store so I could maintain a focus on the code behind the simulation. In this simulation, a flock of kittens (I wanted to use seagulls, but sadly they would only run away) chase a butterfly around a small park scene. The kittens and butterfly are programmed to follow a set path around the area marked with custom path nodes, and avoid certain obstacles marked with custom obstacle objects. In their flocking formation, the kittens use different flocking methods to keep from bumping into each other or venturing off too far from the group. For the observer of the simulation, you have the option of using a first person controller to walk around the scene, or pressing space to switch between multiple different cameras.
