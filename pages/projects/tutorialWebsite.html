TUTORIAL WEBSITE

I created a Minecraft command block tutorial that implements teams for a Web Design class, which required some sort of step-by-step tutorial in a format friendly to different screen resolutions. I chose the Minecraft topic because it was something I was already familliar with, but also could be updated due to changes that had recently occured in the game. I used a div based navigation system, which seemed great at the time, but could have probably been implemented much easier and with more extensibility using a list based approach. I used many images and supporting text to give clear directions to interested users, and tried to keep with a light-colored theme and readable font. Overall, I would say I'm happy with how this project turned out, but I would approach some mobile design choices differently if I were to revisit this project.
